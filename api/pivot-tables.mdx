## Overview

Pivot tables in ClosedXML consist of two cooperating objects:

- **`IXLPivotCache`** — a cached copy of source data (columns become fields). One cache can back multiple pivot tables.
- **`IXLPivotTable`** — the table displayed in a worksheet. It references a cache and declares which fields appear on which axis.

---

## Architecture

```
Source range / table
        │
        ▼
  IXLPivotCache          ← stores field names and cached values
        │
        ├── IXLPivotTable (sheet "Summary")
        └── IXLPivotTable (sheet "Charts")   ← multiple tables can share one cache
```

Pivot cache data are **not** updated automatically when source data change. Call `IXLPivotCache.Refresh()` to synchronise.

---

## IXLPivotCache

### Creating a pivot cache

```csharp
// Via the workbook's PivotCaches collection
IXLPivotCache cache = wb.PivotCaches.Add(range);

// Or implicitly when adding a pivot table directly from a range or table
IXLPivotTable pt = ws.PivotTables.Add("pvt", ws.Cell("H1"), range);
// pt.PivotCache is set automatically
```

### Properties

#### FieldNames

```csharp
IReadOnlyList<String> FieldNames { get; }
```

Names of all fields in source order. Names are unique and case-insensitive. If the source contained duplicate column headers, they are disambiguated by appending a number (e.g. `Cost`, `Cost2`).

---

#### SaveSourceData

```csharp
Boolean SaveSourceData { get; set; }
```

When `true` (default), cached values are serialised into the workbook file. Disable to reduce file size at the cost of requiring the source to be available on open.

---

#### RefreshDataOnOpen

```csharp
Boolean RefreshDataOnOpen { get; set; }
```

When `true`, Excel will refresh the cache when the workbook is opened. Default is `false`.

---

#### ItemsToRetainPerField

```csharp
XLItemsToRetain ItemsToRetainPerField { get; set; }
```

Controls how many stale (deleted) items are kept in each field's distinct-value list after a refresh. See `XLItemsToRetain` below.

---

### Methods

#### Refresh

```csharp
IXLPivotCache Refresh();
```

Re-reads values from the source range/table into the cache. This updates `FieldNames`, removes deleted fields from all dependent pivot tables, and recalculates displayed values.

**Throws** `InvalidReferenceException` if the source range can no longer be found.

---

#### Fluent setters

```csharp
IXLPivotCache SetSaveSourceData();
IXLPivotCache SetSaveSourceData(Boolean value);

IXLPivotCache SetRefreshDataOnOpen();
IXLPivotCache SetRefreshDataOnOpen(Boolean value);

IXLPivotCache SetItemsToRetainPerField(XLItemsToRetain value);
```

---

## IXLPivotCaches

All caches in the workbook are accessible via `XLWorkbook.PivotCaches`:

```csharp
IXLPivotCaches caches = wb.PivotCaches;
IXLPivotCache cache = caches.Add(range);

// Refresh every cache
foreach (var c in wb.PivotCaches)
    c.Refresh();
```

#### Add

```csharp
IXLPivotCache Add(IXLRange range);
```

Adds a new pivot cache for the given range. If the range coincides with an existing table, the cache references the table as its source.

## IXLPivotTable

### Creating a pivot table

```csharp
// Add to a worksheet's PivotTables collection
IXLPivotTable pt = ws.PivotTables.Add("pvt", targetCell, range);
IXLPivotTable pt = ws.PivotTables.Add("pvt", targetCell, table);
IXLPivotTable pt = ws.PivotTables.Add("pvt", targetCell, pivotCache);
```

<ParamField body="name" type="String">
  Unique name of the pivot table. Case-insensitive uniqueness is enforced.
</ParamField>
<ParamField body="targetCell" type="IXLCell">
  Top-left cell of the pivot table output area (filter fields are placed above this cell when present).
</ParamField>

**Throws** `ArgumentException` if a pivot table with that name already exists.

---

### Core identity properties

| Property | Type | Description |
|---|---|---|
| `Name` | `string` | Unique name of the pivot table. |
| `Title` | `string` | Optional display title. |
| `Description` | `string` | Optional description. |
| `TargetCell` | `IXLCell` | Top-left cell of the pivot table. |
| `PivotCache` | `IXLPivotCache` | The data source cache. Can be reassigned. |
| `Worksheet` | `IXLWorksheet` | The worksheet containing this pivot table. |

---

### Field collections

#### RowLabels

```csharp
IXLPivotFields RowLabels { get; }
```

Fields placed on the row axis (vertical). Each distinct value of the field becomes a row group.

#### ColumnLabels

```csharp
IXLPivotFields ColumnLabels { get; }
```

Fields placed on the column axis (horizontal). Each distinct value becomes a column group.

#### Values

```csharp
IXLPivotValues Values { get; }
```

Value fields placed in the data area. See [IXLPivotValue](#ixlpivotvalue).

#### ReportFilters

```csharp
IXLPivotFields ReportFilters { get; }
```

Fields used as page/report filters (displayed above the pivot table).

### XLPivotSubtotals

| Value | Description |
|---|---|
| `DoNotShow` | No subtotals. |
| `AtTop` | Subtotals above group items. |
| `AtBottom` | Subtotals below group items. |

---

### XLPivotLayout

| Value | Description |
|---|---|
| `Compact` | All row fields in one column. |
| `Outline` | Each row field in its own column; items expand below. |
| `Tabular` | Classic tabular layout; each field in its own column. |

---

### XLPivotSortType

| Value | Description |
|---|---|
| `Default` | Manual ordering. |
| `Ascending` | A → Z. |
| `Descending` | Z → A. |

---

### XLItemsToRetain

| Value | Description |
|---|---|
| `Automatic` | Application-determined threshold (default). |
| `None` | Discard unused items immediately on refresh. |
| `Max` | Retain all items until all are unused. |

---

### XLPivotTableTheme

`XLPivotTableTheme` is an enum. Values follow the pattern `PivotStyleLight1`–`PivotStyleLight28`, `PivotStyleMedium1`–`PivotStyleMedium28`, `PivotStyleDark1`–`PivotStyleDark28`, plus `None`.

## Full example

```csharp
using ClosedXML.Excel;

using var wb = new XLWorkbook();
var ws = wb.AddWorksheet();

// Source data
var range = ws.FirstCell().InsertData(new object[]
{
    ("Event",    "Pastry", "Sold", "Cost"),
    ("Charity",  "Pie",    14,     60),
    ("Sampling", "Cake",   8,      80),
    ("Charity",  "Cake",   7,      75),
    ("Charity",  "Pie",    8,      38),
});

// Create pivot table on a new worksheet
var pivotSheet = wb.AddWorksheet("Pivot");
var pt = pivotSheet.PivotTables.Add("SalesSummary", pivotSheet.Cell("A1"), range);

// Row axis: events
pt.RowLabels.Add("Event");

// Column axis: pastry types
pt.ColumnLabels.Add("Pastry");

// Values: sum of Sold, sum of Cost
pt.Values.Add("Sold").SetSummaryFormula(XLPivotSummary.Sum);
pt.Values.Add("Cost").SetSummaryFormula(XLPivotSummary.Sum);

// Layout
pt.Theme = XLPivotTableTheme.PivotStyleMedium9;
pt.SetShowGrandTotalsColumns(false);
pt.SetLayout(XLPivotLayout.Tabular);

// Refresh cache to pick up any data changes
pt.PivotCache.Refresh();

wb.SaveAs("pivot.xlsx");
```
