---
title: "XLWorkbook"
description: "The root object for creating, loading, and saving Excel workbooks (.xlsx, .xlsm). Implements IXLWorkbook."
---

`XLWorkbook` is the entry point for all ClosedXML operations. You create or load a workbook, manipulate worksheets and cells, then save the result.

## Constructors

```csharp
// New empty workbook
var wb = new XLWorkbook();

// Load from file path
var wb = new XLWorkbook("report.xlsx");

// Load from stream
var wb = new XLWorkbook(stream);

// Load with options
var options = new LoadOptions { RecalculateAllFormulas = true };
var wb = new XLWorkbook(options);
```

<Note>
  `XLWorkbook` implements `IDisposable`. Always wrap it in a `using` statement or call `Dispose()` explicitly.
</Note>

## LoadOptions

Control how a workbook is loaded by passing a `LoadOptions` instance to the constructor.

<ParamField body="RecalculateAllFormulas" type="bool" default="false">
  Recalculate all formulas when the workbook is loaded.
</ParamField>

<ParamField body="GraphicEngine" type="IXLGraphicEngine">
  The graphics engine used for this workbook. Falls back to `LoadOptions.DefaultGraphicEngine` when not set.
</ParamField>

<ParamField body="Dpi" type="System.Drawing.Point" default="96, 96">
  DPI used for image sizing and text measurement. Must be positive on both axes.
</ParamField>

<ParamField body="StrictAttributeParsing" type="bool" default="true">
  When `true`, the XML parser throws on invalid attribute values. Set to `false` to tolerate malformed OOXML.
</ParamField>

<ParamField body="CancellationToken" type="CancellationToken">
  Token to cancel formula recalculation triggered during load.
</ParamField>

## SaveOptions

Control how a workbook is saved by passing a `SaveOptions` instance to `Save` or `SaveAs`.

<ParamField body="EvaluateFormulasBeforeSaving" type="bool" default="false">
  When `true`, formula cells are evaluated and the calculated values are written to the file. Evaluation errors are silently skipped.
</ParamField>

<ParamField body="ValidatePackage" type="bool" default="true (Debug) / false (Release)">
  Validate the OOXML package after writing.
</ParamField>

<ParamField body="GenerateCalculationChain" type="bool" default="true">
  Write the calculation chain part to the package.
</ParamField>

<ParamField body="ConsolidateConditionalFormatRanges" type="bool" default="true">
  Merge adjacent conditional format ranges before saving to reduce file size.
</ParamField>

<ParamField body="ConsolidateDataValidationRanges" type="bool" default="true">
  Merge adjacent data validation ranges before saving to reduce file size.
</ParamField>

<ParamField body="FilterPrivacy" type="bool?">
  Sets the `filterPrivacy` flag in the saved workbook. Pass `null` to leave the existing value unchanged.
</ParamField>

## Code examples

### Create and save a workbook

```csharp
using ClosedXML.Excel;

using var wb = new XLWorkbook();
var ws = wb.AddWorksheet("Sales");

ws.Cell("A1").Value = "Product";
ws.Cell("B1").Value = "Revenue";
ws.Cell("A2").Value = "Widget";
ws.Cell("B2").Value = 12500.00;

wb.SaveAs("sales.xlsx");
```

### Load an existing file and modify it

```csharp
using ClosedXML.Excel;

using var wb = new XLWorkbook("report.xlsx");
var ws = wb.Worksheet("Summary");
ws.Cell("A1").Value = "Updated";
wb.Save();
```

### Load from a stream

```csharp
using ClosedXML.Excel;

using var stream = File.OpenRead("data.xlsx");
using var wb = new XLWorkbook(stream);
// workbook is fully loaded; stream can be closed
```

### Save with formula evaluation

```csharp
using ClosedXML.Excel;

using var wb = new XLWorkbook("report.xlsx");
var options = new SaveOptions { EvaluateFormulasBeforeSaving = true };
wb.SaveAs("report-calculated.xlsx", options);
```

### Add a worksheet from a DataTable

```csharp
using ClosedXML.Excel;
using System.Data;

var dt = new DataTable("Products");
dt.Columns.Add("Name");
dt.Columns.Add("Price", typeof(decimal));
dt.Rows.Add("Widget", 9.99m);

using var wb = new XLWorkbook();
wb.AddWorksheet(dt);           // sheet name = "Products"
wb.SaveAs("products.xlsx");
```

### Evaluate a formula at workbook scope

```csharp
using ClosedXML.Excel;

using var wb = new XLWorkbook();
var result = wb.Evaluate("2 + 2 * 3"); // XLCellValue with Number 8
```

### Open a template without saving over it

```csharp
using ClosedXML.Excel;

using var wb = XLWorkbook.OpenFromTemplate("template.xltx");
wb.Worksheet(1).Cell("B2").Value = DateTime.Today;
wb.SaveAs("output.xlsx");
```
