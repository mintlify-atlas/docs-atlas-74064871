---
title: "IXLWorksheet"
description: "Represents a single sheet within an XLWorkbook. Provides access to cells, rows, columns, ranges, and sheet-level settings."
---

`IXLWorksheet` extends `IXLRangeBase`, so every method available on a range is also available directly on a worksheet. You obtain a worksheet reference from `XLWorkbook.AddWorksheet` or `XLWorkbook.Worksheet`.

## Getting a worksheet reference

```csharp
// Add a new worksheet
IXLWorksheet ws = wb.AddWorksheet("Invoices");

// Access by name (case-insensitive)
IXLWorksheet ws = wb.Worksheet("Invoices");

// Access by 1-based position
IXLWorksheet ws = wb.Worksheet(1);

// Safe access without throwing
if (wb.TryGetWorksheet("Invoices", out var ws))
{
    // use ws
}
```

## Sheet visibility

```csharp
public enum XLWorksheetVisibility
{
    Visible,    // Normal, visible tab
    Hidden,     // Hidden, can be unhidden from the UI
    VeryHidden  // Hidden, can only be unhidden programmatically
}
```

```csharp
// Hide and unhide
ws.Hide();
ws.Unhide();

// Set visibility directly
ws.Visibility = XLWorksheetVisibility.VeryHidden;
```

## Cell access

```csharp
// By address string
IXLCell cell = ws.Cell("B3");

// By row and column numbers (1-based)
IXLCell cell = ws.Cell(3, 2);

// By row number and column letter
IXLCell cell = ws.Cell(3, "B");

// By IXLAddress
IXLCell cell = ws.Cell(address);
```

## Range access

```csharp
// By address string — single cell or multi-cell
IXLRange range = ws.Range("A1:C5");

// By first and last address strings
IXLRange range = ws.Range("A1", "C5");

// By corner row/column numbers (1-based)
IXLRange range = ws.Range(1, 1, 5, 3);

// By two cell references
IXLRange range = ws.Range(ws.Cell("A1"), ws.Cell("C5"));

// Multiple ranges
IXLRanges ranges = ws.Ranges("A1:B2,D4:E6");

// Bounding box of all used cells
IXLRange? used = ws.RangeUsed();
IXLRange? used = ws.RangeUsed(XLCellsUsedOptions.AllContents);
```

## Row access

```csharp
// Single row by number
IXLRow row = ws.Row(1);

// All rows
IXLRows rows = ws.Rows();

// Range of rows by number
IXLRows rows = ws.Rows(1, 10);

// Range of rows by string (e.g. "4:5", "7:8,10:11")
IXLRows rows = ws.Rows("1:5");

// Boundary rows
IXLRow first = ws.FirstRow();
IXLRow last  = ws.LastRow();
IXLRow? firstUsed = ws.FirstRowUsed();
IXLRow? lastUsed  = ws.LastRowUsed();

// Used rows with optional filter
IXLRows usedRows = ws.RowsUsed();
IXLRows usedRows = ws.RowsUsed(XLCellsUsedOptions.AllContents);
IXLRows usedRows = ws.RowsUsed(row => row.Cell(1).Value.IsText);
```

## Column access

```csharp
// Single column by number or letter
IXLColumn col = ws.Column(1);
IXLColumn col = ws.Column("A");

// All columns
IXLColumns cols = ws.Columns();

// Range of columns by number
IXLColumns cols = ws.Columns(1, 5);

// Range of columns by letter range or comma-separated
IXLColumns cols = ws.Columns("A:E");
IXLColumns cols = ws.Columns("A", "E");

// Boundary columns
IXLColumn first = ws.FirstColumn();
IXLColumn last  = ws.LastColumn();
IXLColumn? firstUsed = ws.FirstColumnUsed();
IXLColumn? lastUsed  = ws.LastColumnUsed();

// Used columns with optional filter
IXLColumns usedCols = ws.ColumnsUsed();
IXLColumns usedCols = ws.ColumnsUsed(XLCellsUsedOptions.AllContents);
```

## Code examples

### Iterate all used rows

```csharp
using ClosedXML.Excel;

using var wb = new XLWorkbook("data.xlsx");
var ws = wb.Worksheet("Sheet1");

foreach (var row in ws.RowsUsed())
{
    var id   = row.Cell(1).GetValue<int>();
    var name = row.Cell(2).GetValue<string>();
    Console.WriteLine($"{id}: {name}");
}
```

### Copy a sheet to another workbook

```csharp
using ClosedXML.Excel;

using var source = new XLWorkbook("source.xlsx");
using var target = new XLWorkbook();

source.Worksheet("Report").CopyTo(target, "Report");
target.SaveAs("copy.xlsx");
```

### Hide and show a sheet

```csharp
using ClosedXML.Excel;

using var wb = new XLWorkbook("book.xlsx");
var ws = wb.Worksheet("Internal");
ws.Visibility = XLWorksheetVisibility.VeryHidden;
wb.Save();
```

### Freeze panes and set tab color

```csharp
using ClosedXML.Excel;

using var wb = new XLWorkbook();
var ws = wb.AddWorksheet("Dashboard");

ws.SheetView.FreezeRows(1);          // freeze header row
ws.TabColor = XLColor.FromHtml("#94cb04");
wb.SaveAs("dashboard.xlsx");
```

### Get used range

```csharp
using ClosedXML.Excel;

using var wb = new XLWorkbook("data.xlsx");
var ws = wb.Worksheet(1);

var used = ws.RangeUsed();
if (used is not null)
    Console.WriteLine($"Data spans {used.Address}");
```
